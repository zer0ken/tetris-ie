# TETRIS-IE

Single Player Tetris Game Works On Internet Explorer.

## Controls

* In Game
  * Move Right: `Arrow Right`
  * Move Left: `Arrow Left`
  * Rotate Right: `Arrow Up`, `X Key`
  * Rotate Left: `Left Ctrl`, `Z Key`
  * Hold: `Left Shift`, `C Key`
  * Soft Drop: `Arrow Down`
  * Hard Drop: `Space`
* PAUSE: `Esc`
* RESET: `R Key`
